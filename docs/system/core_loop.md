# コアループ

## 1. ギルドで準備

* 依頼を受注する
* ジョブを設定・変更する
* 装備・アイテムを準備する

## 2. ダンジョンへ入る

* 依頼に対応したダンジョンに入る
* 1階層目から探索を開始する

## 3. 探索（階層型）

* ダンジョンは階層構造になっている（1F → 2F → …）

* 各階層で以下が発生：

  * 戦闘
  * アイテム収集
  * リスク判断

* 階層の深さはダンジョンごとに異なる

## 4. プレイヤーの選択

プレイヤーは各階層で以下を判断する：

* 次の階層へ進む
* 帰還する（通常ダンジョンのみ）

## 5. 任意帰還ルール

* 通常ダンジョン：

  * 階層移動時に帰還可能

* 高難易度ダンジョン：

  * 踏破するまで帰還不可

## 6. 依頼の達成条件

### 採取系

* 指定アイテムを指定数集めることで達成
* 敵ドロップも対象になる場合あり

### 討伐系

* 指定モンスターまたはボスを指定数討伐

### 調査系

* 指定階層に到達

### 深層到達

* 最奥階に到達しボスを撃破

## 7. 探索終了条件

以下のいずれかで探索が終了する：

* 依頼達成
* 任意帰還
* 死亡（強制帰還）

## 8. 死亡時の処理

* プレイヤーはダンジョン外へ戻される
* 一部アイテムを失う

## 9. 帰還後

* ギルドに戻る
* 依頼報酬を受け取る
* 成長・強化を行う
* 次の探索に備える

## 10. コアループの目的

* リスクとリターンの選択を繰り返す
* 少しずつ強くなる実感を得る
* より深いダンジョンへ挑戦する
