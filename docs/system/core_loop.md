# コアループ

本作の基本的な進行は以下の循環で構成される。

1. ギルドで情報確認、依頼受注、準備、編成、装備調整を行う
2. ダンジョンへ挑戦する
3. 戦闘、探索、収集、判断を重ねながら進行する
4. 生還して成果物や経験を持ち帰る
5. ギルドで整理、強化、次回準備を行う
6. 冒険者ランクを上げ緊急依頼や新しいダンジョンへ挑戦
7. より深い探索や新たな目標に挑む

## コアループの狙い
- 毎回違う探索体験を生む
- 持ち帰りによる前進感を出す
- ギルドに戻る意味を強くする
- 繰り返し遊ぶ楽しさを作る
