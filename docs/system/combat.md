# 戦闘システム

## 1. 基本方針

本作の戦闘は、不思議のダンジョン系のターン制・グリッド制を採用する。

* フィールドはマス目で構成される
* プレイヤーが1回行動すると、敵も行動する
* 行動の順番と位置取りが重要になる

## 2. 行動単位

以下の行動は基本的に1ターンを消費する：

* 1マス移動
* 通常攻撃
* スキル使用
* 魔法使用
* アイテム使用
* 待機
* 向き変更（要検討）

今後、行動ごとのターン消費量は調整する可能性がある。

## 3. 移動

* プレイヤーはグリッド上を1マスずつ移動する
* 敵も同様にグリッド上を移動する
* マップは部屋と通路を持つダンジョン構造を基本とする

## 4. 通常攻撃

* 通常攻撃は隣接した対象に対して行う
* 基本射程は1マス
* 武器やジョブにより攻撃範囲が変化する可能性がある

## 5. ジョブとの関係

ジョブによって戦闘スタイルが変化する。

例：

* 戦士：近接戦闘に強い
* 魔法使い：遠距離攻撃や範囲攻撃が得意
* モンスター使い：仲間や召喚を活用
* シーフ：機動力や回避、罠対策が得意

詳細は別ファイルで設計する。

## 6. 敵の行動

* プレイヤーが行動した後に敵が行動する
* 敵は視界内のプレイヤーを認識すると接近・攻撃する
* 敵ごとに行動パターンを持つ

## 7. 戦闘の目的

* 位置取りとターン管理を重視する
* リソース消費を考えながら進行する
* 無理に進まず、撤退判断も重要とする
