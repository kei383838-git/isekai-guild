# 絶対ルール

## ゲームの基本
- 本作は Godot で開発するローグライクゲームである
- ゲームタイトルは「異世界の不思議な冒険ギルド」である
- プレイヤーは冒険者ギルドに所属する冒険者で依頼を受けダンジョンを探索する
- ダンジョン探索、持ち帰り、成長の循環をコア体験とする

## 設計方針
- 世界観とゲーム仕様は矛盾しないように設計する
- 新しい案を採用する場合は、既存ルールとの整合性を確認する
- 重要な用語は terminology.md に定義してから使う
- 未確定事項は断定せず、草案として管理する

## ドキュメント運用
- このファイルは最優先の前提とする
- 他の仕様ファイルはこの rules.md に従う
- 他チャットで使用する場合は、必要に応じてこの内容を共有する
- テスト用の追記
